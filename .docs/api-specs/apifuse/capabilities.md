# API Fuse 기능 목록

이 문서는 API Fuse 문서와 README 기준으로 확인된 기능 범위를 기획 참고용으로 정리합니다. 로컬 `openapi.json`은 아직 확보되지 않았으므로 endpoint 단위 확정 전의 카테고리 목록입니다.

## 지원 기능

| 기능 범주 | 가능한 기능 | 주요 입력 | 주요 출력 | 기획 참고 |
| --- | --- | --- | --- | --- |
| 지도/장소 | 장소 검색, 지역/주소 기반 검색, 로컬 POI 조회 | 키워드, 좌표, 지역명 | 장소명, 주소, 좌표, 카테고리 | 주변 후보 탐색, 위치 기반 추천 |
| 경로/이동 | 경로, 내비, 대중교통 관련 조회 | 출발지, 도착지, 이동 조건 | 거리, 시간, 경로 정보 | 동선 비교, 이동 부담 계산 |
| 음식/예약 | 맛집, 음식점, 예약/주문 관련 데이터 조회 | 위치, 키워드, 음식 카테고리 | 매장 정보, 예약/주문 후보 | 식사 후보 수집 |
| 날씨/대기 | 기상청, 에어코리아 기반 날씨/미세먼지 조회 | 지역, 좌표, 날짜/시간 | 날씨, 기온, 강수, 대기 상태 | 실내/실외 판단, 일정 리스크 판단 |
| 쇼핑/가격 | 상품 검색, 가격 비교, 로컬 쇼핑 데이터 조회 | 키워드, 카테고리 | 상품명, 가격, 판매처 | 준비물/구매 후보 추천 |
| 물류/주차 | 배송 추적, 주차장 탐색 | 운송장, 위치 | 배송 상태, 주차장 정보 | 이동 전후 편의 정보 |
| 법률/공공 | 법령 검색, 용어 정의 | 키워드, 법령명 | 법령 본문, 용어 설명 | 일반 리서치/문서 보조 |

## 현재 한계

- 로컬에 API Fuse `openapi.json`이 아직 없습니다.
- 실제 operation 이름, request/response schema, 필수 파라미터는 Playground 또는 API docs에서 추가 확인해야 합니다.
- 서비스별 OAuth가 필요한 operation은 단일 API key만으로 바로 호출되지 않을 수 있습니다.
