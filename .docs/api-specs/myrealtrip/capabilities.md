# Myrealtrip 기능 목록

이 문서는 `.docs/api-specs/myrealtrip-integration.md` 기준으로 확인된 기능 범위를 기획 참고용으로 정리합니다.

## REST 지원 기능

| 기능 범주 | 가능한 기능 | 주요 입력 | 주요 출력 | 기획 참고 |
| --- | --- | --- | --- | --- |
| 항공 공항 검색 | 공항/도시 자동완성, 전체 공항 목록 조회 | 키워드, 정렬, 페이지 | 공항 코드, 도시, 국가 | 항공 검색 전 출도착 공항 선택 |
| 항공 최저가 | 날짜별 최저가, 180일 윈도우, 다중 목적지/전체 목적지 최저가 조회 | 출발 공항, 도착 공항, 기간, 날짜 | 날짜, 항공사, 환승, 총액 | 여행 날짜/목적지 후보 비교 |
| 숙소 지역 검색 | 숙소 검색용 지역 자동완성 | 키워드, 국내/해외 여부 | regionId, 지역명, 타입 | 숙소 검색 전 지역 ID 확보 |
| 숙소 검색 | 지역/체크인/체크아웃/인원 기반 숙소 조회 | regionId, 날짜, 인원, 등급, 페이지 | 숙소 ID, 이름, 가격, 별점, 리뷰, 이미지 | 숙박 후보 수집 |
| 투어/티켓 카테고리 | 도시별 투어/티켓 카테고리 조회 | 도시 | 카테고리 이름, 값 | 액티비티 검색 조건 구성 |
| 투어/티켓 검색 | 키워드/카테고리/가격 기반 상품 검색 | 키워드, 카테고리, 가격, 정렬, 페이지 | gid, 상품명, 가격, 리뷰, URL, 딥링크 | 체험/액티비티 후보 수집 |
| 투어/티켓 상세 | 상품 상세 설명 조회 | gid | 설명, 포함/불포함, 일정, 리뷰 | 후보 상세 비교 |
| 투어/티켓 옵션 | 특정 날짜의 옵션과 실제 가격 조회 | gid, 날짜 | 옵션 ID, 옵션명, 가격, 재고 | 예약 가능 옵션 판단 |
| 투어/티켓 캘린더 | 월별 예약 가능 여부와 시작가 조회 | gid, 월/날짜 | 날짜별 가격, block 여부, 즉시확정 여부 | 일정표에 넣을 수 있는 날짜 확인 |
| 마이링크 | 마케팅 파트너 추적 링크 생성 | targetUrl | mylink, mylinkId | 상품 링크 전환 추적 |
| 항공 랜딩 URL | 항공 운임 조회 랜딩 URL 생성 | 공항, 날짜, 인원, 좌석 등급 | 운임 조회 URL | 항공 검색 결과 딥링크 |
| 수익 현황 | 비항공/항공 수익 조회 | 조회 기간, 기준 | 예약번호, 판매가, 수수료, 정산 상태 | 성과 분석 |
| 예약 내역 | 비항공/항공 예약 조회 | 조회 기간, 상태, 페이지 | 예약번호, 상태, 상품, 가격 | 예약/전환 분석 |
| 상태 확인 | REST 서버 health check | 없음 | `UP` 상태 | 운영 상태 확인 |

## MCP 지원 기능

| 도메인 | 가능한 기능 | 도구 |
| --- | --- | --- |
| 숙소 | 숙소 검색, 숙소 상세/객실/리뷰/편의시설 조회 | `searchStays`, `getStayDetail` |
| 항공 | 국내선/국제선 항공편 검색, 프로모션 항공사 조회, 날짜별 최저가 캘린더 조회 | `searchDomesticFlights`, `searchInternationalFlights`, `getPromotionAirlines`, `flightsFareCalendar` |
| 투어/액티비티 | 상품 검색, 상세 조회, 옵션 조회, 카테고리 조회 | `searchTnas`, `getTnaDetail`, `getTnaOptions`, `getCategoryList` |
| 공통 | 현재 한국 시간 조회 | `getCurrentTime` |

## 현재 한계

- 로컬 OpenAPI 파일은 없습니다.
- REST는 Bearer API key가 필요합니다.
- MCP는 성과 트래킹 없이 AI 도구에서 직접 탐색할 때 쓰는 용도로 정리되어 있습니다.
- 일부 API는 rate limit이 낮으므로 서버 측 throttle과 캐시가 필요합니다.
