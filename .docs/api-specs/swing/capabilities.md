# Swing 기능 목록

이 문서는 `openapi.json` 기준으로 확인된 기능 범위를 기획 참고용으로 정리합니다.

## 지원 기능

| 기능 범주 | 가능한 기능 | 주요 입력 | 주요 출력 | 기획 참고 |
| --- | --- | --- | --- | --- |
| 주변 기기 조회 | 좌표 중심 반경 안의 공유 모빌리티 기기 목록 조회 | 위도, 경도, 반경, 개수 | 기기 목록, 위치, 상태 정보 | 주변 이동 수단 후보 표시 |
| 택시 ETA 조회 | 출발지-도착지 택시 예상 거리/시간/통행료/요금 조회 | 출발 좌표, 도착 좌표 | 이동 거리, 소요 시간, 통행료, 택시 요금 | 이동 비용/시간 비교 |

## 현재 한계

- OpenAPI에는 2개 path만 있습니다.
- `radius`는 1m부터 10km까지, `count`는 1부터 100까지 제한됩니다.
- stage 환경 응답은 mock 값일 수 있습니다.
- 인증은 `X-API-KEY`가 필요하며 키는 커밋하지 않습니다.
