# Rocketpunch 기능 목록

이 문서는 `openapi.json` 기준으로 확인된 기능 범위를 기획 참고용으로 정리합니다.

## 지원 기능

| 기능 범주 | 가능한 기능 | 주요 입력 | 주요 출력 | 기획 참고 |
| --- | --- | --- | --- | --- |
| 게시글 조회 | 게시글 검색/목록, 게시글 상세 조회 | 검색 조건, page, postId | 게시글 목록, 게시글 상세 | 커뮤니티/콘텐츠 탐색 |
| 게시글 작성/관리 | OAuth 사용자 컨텍스트로 게시글 작성, 수정, 삭제 | 본문 JSON, 이미지, postId | 생성/수정/삭제 결과 | 사용자 계정 연동 기능 |
| OAuth 인증 | Authorization Code + PKCE 인가, token 발급/갱신, token 폐기 | client, code, verifier, refresh token | access token, refresh token | 사용자 컨텍스트 API 호출 |
| 채용공고 조회 | 채용공고 검색/목록, 상세 조회 | 키워드, 직군, 산업, 숙련도, 고용형태, jobId | 채용공고 목록, 상세, 모집 상태 | 커리어/채용 데이터 탐색 |
| 이벤트 조회 | 이벤트 검색/목록, 상세 조회 | 키워드, 카테고리, 주제, 지역, 기간, 상태, eventId | 이벤트 목록, 상세 | 행사/네트워킹 정보 탐색 |
| 이벤트 부가 정보 | 이벤트 티켓, 세션, FAQ 조회 | eventId | 티켓 목록, 타임테이블, FAQ | 이벤트 상세 화면 구성 |
| 코드 목록 | 채용/이벤트 필터용 코드 조회 | 없음 | 숙련도, 산업, 직군, 주제, 카테고리, 고용형태 코드 | 검색 필터 UI 구성 |

## 현재 한계

- 일부 쓰기 기능은 OAuth와 `post:write` scope가 필요합니다.
- SAI 핵심 도메인과 직접 연결되지는 않아 후순위 리소스로 분류되어 있습니다.
- 행사 환경에서는 운영팀이 배포한 `X-OBA-API-Key` 사용이 전제입니다.
